git

git是一个开源的分布式版本控制软件,有别于CVS和SVN等集中式版本控制系统.

git的发明人是Linux教父Linus Torvalds.

git诞生过程大事记:

  • 2005年4月3日,开始开发git.
  • 2005年4月6日,项目发布.
  • 2005年4月7日,git就可以作为自身的版本控制工具了.
  • ...
  • 2008年4月10日,GITHub正式发布.

GitHub

GitHub是一个面向开源及私有软件项目的托管平台,因为只支持git作为唯一的版本库格式进行托管,故名GitHub.

截止到2015年,GitHub已经有超过 12.1M 注册用户和 30.6M 代码仓库. 事实上已经成为了世界上最大的代码存放网站.Github是目前全球最大的男性同性交友平台.

GitHub的吉祥物"章鱼猫"(英语:Octocat)

GitHub

GitHub提供了 Web界面 ,你可以在上面创建资源仓库来存放你的项目.

在本地或服务器创建一个资源仓库通过 Shell命令 或图形用户界面可以和远端的GitHub进行项目同步更新,实现对项目的管理.

GitHub亮点

  1. 只支持git
  2. 社交化编程
  3. 创建个人主页或项目主页

GitHub图形用户界面缺点

  1. 安装
  2. 界面简陋
  3. 启动速度慢
  4. 缺少其他辅助工具
